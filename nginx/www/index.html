Features

This server is running Vanilla Java Minecraft version 1.18.

Texturepack

This server uses the beautiful johnsmithlegacy texture pack and will prompt you to install this pack after joining the server.

For further information take a look at https://www.johnsmithlegacy.co.uk.

Download Mirror: JohnSmithLegacy 1.18.1 v4  SHA1:8c5b86da9e6667f862ce45f3c0d52bbd37ff4570

Joining

By far the easies way to manage and control minecraft instances is to use MultiMc. Just create a new 1.18 instance and you are ready to go.

This server uses Whitelisting, please contact the Adminitrator with you profile name and UUID. In case you do not know your UUID you can use a tool like https://minecraftuuid.com/ find it.

Download Mirror: MultiMc 5 0.6.14 Windows  SHA1: caa57641222e486e4bf1bd7d4d196ce13078fb8d
